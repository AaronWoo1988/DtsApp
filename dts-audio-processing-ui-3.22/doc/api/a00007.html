void com.dts.dtssdk.accessory.Brand.getImageBytes ( Context  context,
int  imageSizeCategory,
boolean  fallbackToOtherSizes,

Loads an image from the datastore, based on the image size category. Once the datastore loads the image, the callback will be called with the byte array of the image.

Parameters
contextContext of the application
imageSizeCategorysize category to load. AccessoryIcon.IMAGE_LARGE will load the large image associated with the Brand AccessoryIcon.IMAGE_MEDIUM will load the medium image associated with the Brand AccessoryIcon.IMAGE_SMALL will load the small image associated with the Brand
fallbackToOtherSizeswhether or not the function should attempt to load other sizes if the requested size is not available
loadImageCallbackcallback to be called when the image has finished loading
