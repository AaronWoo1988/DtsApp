Detailed Description

Main Class for the DTS SDK

DtsManager provides all audio features of DTS API and all calls to get and set various DTS audio settings should be made via this class. As DtsManager is designed to be a singleton class, the getInstance() function will return a singleton DtsManager instance that can be used to access the API functions. The getInstance() function and DtsManager class are thread-safe.

Note
Some of the features and/or routes might not be licensed for the level of this SDK.
See com.dts.dtssdk.DtsFeature and com.dts.dtssdk.util.AudioRoute for what is licensed and not for this particular level of the SDK.
Only functions and classes for licensed features are documented here, and others are hidden.
Although unlicensed features are exposed by the API, calling them will result in an error (FEATURE_NOT_LICENCED).
At runtime you can use the com.dts.dtssdk.DtsFeatureChecker to query what features and routes are available.
Warning
Documented return values are not necessarily an exhaustive list of possible return codes but limited to the SDK and the Service layers, and not errors resulting from lower layers such as the driver and the "eagle" layers.
