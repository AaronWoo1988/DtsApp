DTS Audio Processing SDK Documentation

INTRODUCTION

  • This documentation includes the definitions of classes for the DTS Audio Processing System Solution for Android.
  • The first main class that is of particular interest to developers is com.dts.dtssdk.DtsManager, through which most interaction with the SDK is done.
  • The other main class of interest is com.dts.dtssdk.accessory.AccessoryDatastore, which is used you query and retrieve accessories etc from the database, for passing to com.dts.dtssdk.DtsManager.
Note
Some of the features and/or routes might not be licensed for the level of this SDK.
See com.dts.dtssdk.DtsFeature and com.dts.dtssdk.util.AudioRoute for what is licensed and not for this particular level of the SDK.
Only functions and classes for licensed features are documented here, and others are hidden.
Although unlicensed features are exposed by the API, calling them will result in an error (FEATURE_NOT_LICENCED).
At runtime you can use the com.dts.dtssdk.DtsFeatureChecker to query what features and routes are available.
